本地分支落后远程分支

发表于2023-01-04|更新于2023-07-31|git
|字数总计:47|阅读时长:1分钟|阅读量:

问题：
终端提示：更新被拒绝，因为您当前分支的最新提交落后于其对应的远程分支。

两步：

  1. git fetch origin
  2. git merge origin/目标分支
